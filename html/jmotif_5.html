cosine_sim R Documentation

Computes the cosine distance value between a bag of words and a set of TF-IDF weight vectors.

Description

Computes the cosine distance value between a bag of words and a set of TF-IDF weight vectors.

Usage

cosine_sim(data)

Arguments

data

the list containing a word-bag and the TF-IDF object.
