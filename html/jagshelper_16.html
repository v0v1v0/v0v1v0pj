expit R Documentation

Expit, or inverse logit

Description

Inverse logit, where logit is defined as log(x/(1-x)).

Expit (inverse logit) is defined as exp(x)/(1+exp(x)).

Usage

expit(x)

Arguments

x

Numeric vector

Value

Numeric vector

Author(s)

Matt Tyers

See Also

logit

Examples

expit(0)
