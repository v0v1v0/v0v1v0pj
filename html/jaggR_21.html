Tidy BUGS files

Description

This function cleans up the formatting

Usage

tidy_bugs(path = ".", arrow = TRUE, brace.newline = FALSE,
  indent = 2)

Arguments

path

location of file(s)
arrow

use the <- operator if TRUE, = otherwise.
brace.newline

move braces to a new line if TRUE
indent

number of spaces to indent code blocks
