minify_json R Documentation

Minify Json

Description

Removes indentiation from a JSON string

Usage

minify_json(json, ...)

Arguments

json

string of JSON
...

other argments passed to to_json

Examples


df <- data.frame(id = 1:10, val = rnorm(10))
js <- to_json( df )
jsp <- pretty_json(js)
minify_json( jsp )
