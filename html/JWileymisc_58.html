intSigRegGraph R Documentation

Function to find significant regions from an interaction

Description

This function uses the contrast function from rms to find the threshold for significance from interactions.

Usage

  contrastList,

  varyvar,
  varyvar.levels,

  ratio = 1,
  xlim,
  ylim,
  xbreaks,
  xlabels = xbreaks,
  scale.x = c(m = 0, s = 1),
  scale.y = c(m = 0, s = 1),
  starts = 50
