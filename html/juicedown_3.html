anki_setup R Documentation

Hide/Show text

Description

This function inserts JavaScript code for anki texts. Call this function in a code chunk somewhere in your R Markdown file. Use anki() function with a inline r code, to make a hidden block.

Usage

anki_setup(
  background = "yellow",
  color = "blue",
  border = "slateblue",
  padding = 3,
  margin = 3,
  cursor = "pointer"
)

Arguments

background

character. Background color of the blank box.
color

character. Font color of the answer of the blank box.
border

character. Border color of the blank box.
padding

integer. Space between the answer text and the border.
margin

integer. Left and right margin around the border.
cursor

character.

Value

character. CSS/JavaScript code for anki texts.
