jot_list R Documentation

List out names of values in a jot notepad

Description

List out names of values in a jot notepad

Usage

jot_list(pad = jot_active())

Arguments

pad

path to notepad to open

Value

vlist of names of values in notepad

Examples

tf <- tempfile()
jot_new_pad(tf)
jot_activate(tf)
jot('ex_val', 'ex')
jot_list()
