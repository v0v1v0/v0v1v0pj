display_jcolors R Documentation

Display jcolors

Description

displays the discrete jcolors palettes

Usage

display_jcolors(
  palette = c("default", "pal2", "pal3", "pal4", "pal5", "pal6", "pal7", "pal8", "pal9",
    "pal10", "pal11", "pal12", "rainbow")
)

Arguments

palette

Character string indicating a palette of colors.

Examples

display_jcolors()
