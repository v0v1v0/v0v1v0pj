jot_erase R Documentation

Erase a Note

Description

Erase a Note

Usage

jot_erase(name, pad = jot_active())

Arguments

name

character, name of note to erase. required.
pad

path to file of pad to write to. default is jot_active().

Value

path to pad, invisibly

Examples

tf <- tempfile()
jot_new_pad(tf)
jot(3, 'example')
jot_erase('example')
