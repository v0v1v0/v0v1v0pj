nparam R Documentation

Number of parameters

Description

Total number of individual parameter nodes saved in jagsUI output.

Usage

nparam(x)

Arguments

x

Output object from jagsUI::jags()

Value

A single numeric value giving the number of parameter nodes.

Author(s)

Matt Tyers

Examples
