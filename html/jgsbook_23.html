ztrans R Documentation

z-Transformation by given numbers, with z = (x - mu) / sd

Description

z-Transformation by given numbers, with z = (x - mu) / sd

Usage

ztrans(x, mu = 0, sd = 1)

Arguments

x

a value to transform
mu

the given mu
sd

the given standard deviation

Value

the z-transformed value

Examples

ztrans(120,mu=118,sd=20)
